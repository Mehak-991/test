# 📨 Premium Contact Us Form

A sophisticated, recruiter-ready "Contact Us" form featuring a modern glassmorphism design, smooth micro-interactions, and robust API integration.

## ✨ Project Highlights

- **Luxury Design Language**: Implemented using Vanilla CSS with `backdrop-filter` for a sleek glass effect, curated typography (Playfair Display & Outfit), and vibrant background asets.
- **Dynamic User Experience**: 
  - Real-time loading states (spinner) on the submit button.
  - Interactive success/error notifications.
  - Seamless responsive layout for all device sizes.
- **API Integrated**: Fully functional `POST` request to the Great Frontend API, sending data in the required JSON format.

## 🛠️ Technical Specifications

- **Frontend**: HTML5, Vanilla CSS3 (Custom Variables), JavaScript (ES6+).
- **API Endpoint**: `https://www.greatfrontend.com/api/questions/contact-form`
- **Data Format**: 
  ```json
  {
    "name": "User Name",
    "email": "user@example.com",
    "message": "User's detailed message"
  }
  ```

## 📂 File Structure
- `index.html`: Semantic structure of the application.
- `style.css`: The "Premium" design system.
- `script.js`: Submission logic and API handling.
- `launch_sandbox.html`: Automation tool for cloud deployment.
- `contact_us_background.png`: Custom AI-generated background asset.
